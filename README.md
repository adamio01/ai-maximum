# AI Архитектор — landing page

Готовый прототип сайта в тёмной палитре с синими акцентами, 3D-визуалом и адаптивной структурой.

## Файлы
- `index.html` — верстка лендинга
- `styles.css` — стили и тема
- `script.js` — интерактивное 3D-движение и появление секций

## Как использовать
1. Открой `index.html` в браузере.
2. Размести сайт на любом статическом хостинге: Vercel, Netlify, GitHub Pages.
3. Оплата будет в связке с Prodamus: сайт только ведёт к оформлению, без собственных popup-checkout.

## Рекомендации по архитектуре
- Статический фронтенд + CDN для скорости и защиты.
- Backend для защиты данных и обработки webhooks: API на Next.js / Node.js / Flask.
- HTTPS, CORS, CSP, rate limiting и WAF для защиты от DDoS и взлома.
- Сервисы: Cloudflare, Fastly, AWS WAF.

## Безопасность
- Не хранить платёжные данные на фронтенде.
- Основной вариант оплаты — Prodamus, с передачей клиента на внешнее оформление.
- Защищать формы с помощью серверной валидации и повторной проверки токенов.
- Отдельная подсеть/сервер для API и webhook-обработки.
